# Legislative Handbook - Document Search

A high-fidelity, three-pane website prototype for legislative staff to search and view legislative documents including bills, hearing questions, and committee information.

## Features

### Layout
- **Top Search Bar**: Global search with live preview for quick document discovery
- **Left Scrollable Nav**: Organized list of reports, bills, hearings, and committees with filtering
- **Right Content Pane**: Detailed view with Bill #, Text, Official Response, Notes, and Data Tables

### Key Data Types
- **Bills**: Full bill text, financial impact tables, voting records, official responses, and staff notes
- **Hearing Questions**: Hearing summaries, questions raised, witness testimony tables
- **Committee Info**: Committee overview, jurisdiction, member lists, active legislation

### Design Features
- Rich text support for legislative documents
- Legible data tables for high-density information display
- Professional, authoritative styling suitable for legislative staff
- Responsive design for various screen sizes
- Print-friendly styling

## Getting Started

1. Open `index.html` in a web browser
2. Use the search bar to find specific documents
3. Click on items in the left navigation to view details
4. Use filter buttons to narrow down document types
